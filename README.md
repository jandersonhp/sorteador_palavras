## Sobre o Projeto: 
Estou criando esse projeto em python totalmente do 0, sem ajuda de IA, sem ajuda de Google, fóruns, ou qualquer código pronto. Usando apenas o pouco que aprendi de Python e a documentação dentro da IDLE do Python 3.

## Como funciona o Projeto:
Comecei esse projeto no ambiente de trabalho em meu tempo ocioso, porém as máquinas do serviço são bem limitadas, não dá pra abrir nenhum site externo (apenas intranet), então tenho que fazer tudo sem muita ajuda, apenas raciocínio, lógica e pesquisa dentro do F1 -documentação- da IDLE.
Não consigo exportar o código -não podemos ter nada de anotações, nem enviar e-mail, nem usar o celular na operação-, então decidi tentar lembrar de tudo e reescrever do 0 ao chegar em casa, por isso estou montando esse repositório e vai levar bastante tempo pra eu conseguir atualizar todo o progresso aqui.

## Contexto do Projeto
A ideia desse projeto veio de um dia quando uma colega de trabalho estava comentando que estava em dúvida de onde almoçar e ela mesma se deu algumas sugestões "pizza hut... mcdonalds... burger king... self service..." e eu brinquei "se tá indecisa, sorteia um lugar, ué rs", aí um dia (dia 31/12/2025) por ser virada de ano, tive bastante tempo ocioso na empresa, de repente lembrei dessa interação com a colega de trabalho e pensei "e se eu criar um programa pra sortear o local, então?". Daí tive a ideia de escrever em Python (única ferramenta disponível na máquina) pra pedir as palavras e então sorteá-las.

## Por que o projeto é simples
Este projeto é propositalmente simples. O objetivo principal não é criar uma aplicação complexa,
mas consolidar fundamentos da linguagem Python, como lógica de programação, organização de código
e resolução de problemas em um ambiente com restrições.
